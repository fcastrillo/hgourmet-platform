# FEAT-3: Canal de Comunicacion y Conversion WhatsApp

## Benefit Hypothesis

- **Para**: clientes locales y recurrentes de HGourmet que navegan el catalogo desde movil o desktop
- **Que**: necesitan contactar a la tienda sin friccion para resolver dudas y confirmar pedidos
- **Esta Feature**: integra puntos de contacto persistentes (boton flotante, CTA contextual y pagina de contacto) con paridad visual respecto al prototipo validado
- **Esperamos**: aumentar conversion de consulta a pedido y reducir abandono por falta de canal de contacto inmediato
- **Sabremos que hemos tenido exito cuando**: logremos al menos 50 pedidos confirmados via WhatsApp originados desde la web en el primer mes
- **Hipotesis (formato constitucion)**: Si entregamos puntos de contacto persistentes de WhatsApp en todo el storefront, entonces los clientes podran iniciar conversaciones y confirmar pedidos con menor friccion, medido por al menos 50 pedidos confirmados via WhatsApp originados desde la web en el primer mes.
